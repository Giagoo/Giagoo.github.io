索引

记录的操作包括：视图、选择、模型、物体、UV、灯光、动画、雕刻、CG流程

视图

移动视图：Ctrl+鼠标中键
旋转视图：鼠标中键
前/左/顶视图： 1/3/7
后/右/底视图: Ctrl+1/3/7
进入/退出摄像机视角：0
正交/透视视图 ： 5
聚焦已选中物体： .（小键盘）
Home： 所有框架

左键移动3D浮标
Shift+C： 重置3D游标的位置
/（小键盘）： 隔离并聚焦已选中框架
Z： 线框/物体模式显示
Alt+Z： 贴图/物体模式显示
Shift+Z： 预渲染/物体模式显示

选择

右键选择物体
Tab键切换编辑/物体模式
A全选（物体模式）/取消选择
B+左键： 矩形选择工具（加选） B+中键： 矩形选择工具（减选）
C+左键： 拖拽选择工具（加选） C+中键： 拖拽选择工具（减选）
拖拽和矩形选择工具皆可滑动滚轮改变笔刷大小
Ctrl+左键: 套索选择工具（加选） Ctrl+shift+左键 ：套索选择工具（减选）
Alt+右键： 选择循环边/面/点
Ctrl+Alt+右键： 选择环形边
Ctrl+Tab： 切换点/边/面选择模式
Ctrl+（+/-）： 扩大/缩小选择范围
选择共享的点/边/面

模型

环切： Ctrl+R，滚动滚轮添加段数,左键移动循环边，右键释放环形边至中心
细分： L键选择物体，W调出专用项，选择细分命令，T调出工具栏选择细分次数
多边形切割工具： K
重复上一次操作：Shift + R
内切面： I （内切后按住Ctrl可以控制切面挤出的高度）
挤出： E
倒角： Ctrl+B，滚动滚轮添加段数
G + G： 滑动点/线/面（不改变物体形状）
撕裂同时填补： Alt + V（光标的位置很重要）
补洞/连接： F
衰减编辑： O
相连项衰减编辑： Alt + O
应用比例/缩放： Ctrl + A
F6：调出上一步命令的属性
Ctrl + Alt + 空格 ： 确定选择元素为新坐标
Ctrl + B ： 框选渲染预览框（需在物体模式下） (Alt + Ctrl + B取消框选)
Shift + Tab : 打开吸附
Shift + E ： 折痕
Ctrl + M ： 镜像（先shift + D复制，再Ctrl + M，再按xyz确定轴向）
Ctrl + T ： 三角化

物体

添加项目： Shift + A
移动： G （按住Ctrl键可以吸附网格移动）
缩放： S
旋转： R
恢复旋转/缩放/移动： Alt + R/S/G
在XYZ轴上移动： G + X/Y/Z （在X/Y/Z轴外移动： G +Ctrl + X/Y/Z），旋转缩放同理
移动五个单位： G + 5
缩放两倍： S + 2
旋转30°： R + 30
搜索菜单： 空格
设置原点： Ctrl + Shift + Alt + C
吸附光标/选择物体： Shift + S
复制: Shift + D
连接复制： Alt + D（不可复制面）
撕裂点： V
分离点/线/面： P
合并项目： Ctrl + J
合并点线面： Alt + M
反选： Ctrl + I
设置父级： Ctrl + P（后选为父）
清空父级： Alt + P
移动物体至另一图层： M
多个图层显示： Ctrl + 左键点选要显示的图层

UV

标记缝合边：在边模式下选择边，Ctrl + E键选择标记缝合边命令

动画

播放/暂停： Alt + A
反向播放/暂停： Shift + Alt + A
上/下一帧： 左/右箭头
向前/后跳十帧： Shift + 上/下箭头
回到开始/结束帧： Shift + 左/右箭头

雕刻

插入图片：N键打开属性栏，勾选背景图，点击打开按钮，参考图只出现在正顶视图下
开启动态拓扑： 在雕刻模式下按Ctrl + D
控制笔刷大小： F
控制笔刷力度： Shift + F
Ctrl + I ： 反转遮罩

灯光

渲染模式下面片光源的隐藏： 物体——>Cycles——>关闭摄像机摄像机

CG流程

Sculpting或Hard Surface(模型面数：游戏：1000——2500，影视：100000——300000)
Retopology(重拓朴)
Uvwrap(展开Uv)
Texture(贴图)
Rig(绑定)
Skin(权重绘制)
Animation(动画)
Composite(合成)
