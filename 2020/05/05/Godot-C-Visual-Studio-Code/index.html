Godot + C# + Visual Studio Code

Posted on 2020-05-05 Edited on 2021-01-05

Dependencies

Godot

VSCode

VSCode Extensions

  • C#

  • Mono Debug

  • Godot-tools

  • 如果VSCode提示无法链接至GDscript，因为Godot未启动的原因，启动Godot可以解决这个问题

Visual Studio

  • In Visual-studio, make sure you have universal windows platform development and .NET desktop development and underneath .NET Framework 4.7 development tools

配置

  1. 启动VSCode，打开Godot项目，按f1键，输入Godot，选择Godot Tools:Open workspace with Godot editor,在弹出的窗口中选择Godot.exe

  2. 创建一个新的场景，拖入一个元素在场景中，Running

  3. 打开Project Setting
